# 🛒 Customer Purchase Behavior Analysis

## 📌 Project Overview
This project analyzes customer shopping behavior using Python, SQL, and Power BI. The objective is to identify patterns, trends, and key factors influencing customer purchasing decisions.

---

## 📊 Dataset Summary
The dataset contains customer transaction and demographic details.

### Key Features
- Customer_ID
- Age
- Gender
- Category
- Purchase_Amount
- Frequency_of_Purchases
- Payment_Method
- Review_Rating
- Subscription_Status
- Discount_Applied
- Season
- Size
---

## 🛠️ Tools Used
- Python (Pandas)
- PostgreSQL
- SQL
- Power BI

---

## 🔄 Data Processing
- Loaded dataset using Pandas  
- Checked missing values  
- Cleaned and standardized data  
- Renamed columns  

---

## ⚙️ Feature Engineering
- Created **Age_Group** column
- Created **purchase_frequency_days** column  
- Converted purchase frequency into days  
- Removed redundant columns  

---

## 🗄️ Database Integration
- Connected Python with PostgreSQL  
- Loaded dataset into database  

---

## 🧮 Data Analysis using SQL
Used SQL queries to analyze customer data:

- Revenue by Age Group  
- Purchase Frequency by Gender  
- High-Value Customers  
- Revenue by Category  
- Seasonal Trends  
- Popular Product Size  
- Payment Method Analysis  

---

## 📊 Dashboard
Created a Power BI dashboard to visualize:
- Revenue analysis  
- Customer segmentation  
- Sales trends

## 🔍 Key Insights
- Young and adult customers generate more revenue  
- Footwear category performs best  
- Summer season has higher sales  
- Digital payments show higher spending
